# About
This was a Visual Lift project where we made a visualization of [Alfredsson Transports](https://www.alfredssontransport.com/) new sorting system for logistical centers. 

# Project goal
The goal with the visualization was to show how the new application and logistical sorting system that Alfredsson Transport had implemented could save both time and help the environment by enabling more efficient truck use for cargo distribution in a city. 

# 3D content
The environment was based on content from [NVIDIA Omniverse](https://www.nvidia.com/en-us/omniverse/) and was brought into [Unreal Engine 5](https://www.unrealengine.com/)e using the [USD](https://dev.epicgames.com/documentation/unreal-engine/universal-scene-description-usd-in-unreal-engine) pipeline. Some of the content, such as the cargo, was hand made in Blender. The trucks in the visualization was purchased from a 3D vendor.
<Image src="Images/cargo.png" width="100%" />

# Visualization
The visualization shows both the current state of cargo handling, and how their system makes things smoother and more organized. The animations where all made by hand using [Sequencer](https://dev.epicgames.com/documentation/unreal-engine/cinematics-and-movie-making-in-unreal-engine), and rendered with [Movie Render Queue](https://dev.epicgames.com/documentation/unreal-engine/rendering-high-quality-frames-with-movie-render-queue-in-unreal-engine). Editing and final render was done in [Adobe After Effects](http://adobe.com/products/aftereffects).
